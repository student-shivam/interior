# Krishna Interior House - Premium Interior Design Website

A modern, luxury, fully responsive interior design company website built with HTML5, CSS3, and Vanilla JavaScript.

## 🎨 Features

### Design & UI
- **Luxury Modern Design** - Elegant and premium interface inspired by high-end interior design companies
- **Fully Responsive** - Optimized for mobile, tablet, and desktop screens
- **Glassmorphism Effects** - Modern glass-effect cards and components
- **Smooth Animations** - Fade-in, scroll animations, hover effects, and transitions
- **Professional Color Palette** - White, black, beige, wood, and gold colors

### Pages
1. **Home** - Hero section with CTA buttons, services preview, gallery preview, testimonials, and stats
2. **About Us** - Company story, mission, vision, team, and detailed "Why Choose Us" section
3. **Services** - Complete list of 12+ services with descriptions and benefits
4. **Gallery** - Filterable portfolio with lightbox preview
5. **Blog** - Blog posts with categories and pagination
6. **Contact Us** - Contact form, business info, map, and FAQ section

### Interactive Features
- **Responsive Navbar** - Sticky navbar with hamburger menu for mobile
- **Hero Image Slider** - Auto-rotating hero section with dark overlay
- **Gallery Filter** - Filter gallery by category (Living Room, Bedroom, Office, Kitchen, Ceiling Design)
- **Lightbox Preview** - Click images to view in fullscreen with arrow navigation
- **Animated Counters** - Numbers animate when section comes into view
- **FAQ Accordion** - Expandable FAQ items with smooth transitions
- **Contact Form** - Functional form with validation
- **Newsletter Signup** - Email subscription form
- **Floating WhatsApp Button** - Fixed WhatsApp chat button
- **Scroll to Top Button** - Quick navigation to top of page
- **Smooth Scrolling** - Smooth scroll navigation throughout the site

### Sections
- Hero Section with 3 CTA buttons
- Premium Services Preview (6 featured services)
- Why Choose Us (6 feature cards)
- Statistics Section (500+ projects, 300+ clients, etc.)
- Gallery Preview with overlay effects
- Testimonials Section with star ratings
- Contact CTA Section
- Footer with links and social media

```css
    /* ... more colors ... */
}
```
